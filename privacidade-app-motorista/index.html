Política de Privacidade — Forbos Motorista

Última atualização: 8 de abril de 2026

Esta Política de Privacidade descreve como o aplicativo Forbos Motorista ("o App") coleta, usa, armazena e protege as informações dos motoristas que o utilizam durante suas viagens. O App é de uso exclusivo de motoristas vinculados a transportadoras clientes do ERP Forbos Logística ("Forbos" ou "nós").

1. Quem é o controlador dos dados

O controlador dos dados tratados pelo App é a Forbos Tecnologia LTDA (CNPJ 30.554.135/0001-60), em conjunto com a transportadora empregadora do motorista. A transportadora é a responsável primária pelo cadastro do motorista no sistema e pelo tratamento dos dados operacionais da viagem. A Forbos atua como operadora, fornecendo a plataforma tecnológica.

2. Quais dados coletamos

2.1 Dados de identificação

  • Nome completo do motorista (já cadastrado na transportadora)
  • CPF — utilizado como login e para identificação única
  • Telefone celular — utilizado para envio do código de verificação por WhatsApp

2.2 Dados do dispositivo

  • Identificador único do dispositivo (device ID) — para autenticação do aparelho
  • Modelo e plataforma (Android/iOS) — apenas para suporte técnico

2.3 Dados operacionais da viagem

  • Dados da viagem atribuída (origem, destino, status, documentos vinculados)
  • Respostas dadas a checklists operacionais (OK/NOK, observações)
  • Eventos registrados durante a viagem (coleta, entrega, ocorrências)
  • Despesas lançadas (tipo, valor, data)

2.4 Dados sensíveis do contexto operacional

  • Localização precisa (GPS) — coletada no momento em que o motorista registra um evento ou uma despesa, para fins de georreferenciamento do lançamento
  • Fotografias — capturadas pelo motorista como comprovantes de checklist, eventos de entrega e despesas
  • Nome e documento de quem recebe a mercadoria (quando se aplica a um evento de entrega)

4. Com quem compartilhamos os dados

Nós não vendemos seus dados. O compartilhamento é limitado ao estritamente necessário:

  • Com sua transportadora empregadora: todos os dados operacionais coletados pelo App são disponibilizados no ERP da transportadora a que o motorista está vinculado. Essa é a razão de existir do App.
  • Com provedores de infraestrutura: usamos servidores próprios hospedados em VPS no Brasil para processar e armazenar os dados. Em nenhum momento os dados transitam por serviços fora do nosso controle.
  • Com o Meta/WhatsApp: exclusivamente o número de telefone é compartilhado para envio do código de verificação. Nenhum outro dado é transmitido via WhatsApp.
  • Por obrigação legal: caso determinado por autoridade competente (fiscalização, ordem judicial, etc.).

5. Onde e por quanto tempo guardamos

Os dados são armazenados em servidores localizados no Brasil. A retenção segue as regras fiscais e trabalhistas aplicáveis à transportadora empregadora — tipicamente 5 anos para dados operacionais (documentos fiscais, eventos de entrega) e pelo período do contrato de trabalho para dados do motorista.

No dispositivo do motorista, o App mantém um cache local (SQLite criptografado e tokens em secure storage) para permitir o funcionamento offline. Esse cache é apagado no logout.

6. Segurança

  • Toda a comunicação entre o App e nossos servidores usa HTTPS/TLS 1.2+
  • Senhas e tokens são armazenados em storage criptografado do sistema operacional (Android Keystore / iOS Keychain)
  • Sessões expiram automaticamente e exigem reautenticação
  • O acesso é vinculado ao dispositivo: cada celular precisa ser autorizado via código por WhatsApp antes do primeiro uso

7. Seus direitos (LGPD)

Como titular de dados, você pode, a qualquer momento, solicitar:

  • Confirmação da existência de tratamento dos seus dados
  • Acesso aos dados que temos sobre você
  • Correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação de dados desnecessários
  • Portabilidade para outro fornecedor
  • Eliminação dos dados tratados com base no seu consentimento
  • Informação sobre compartilhamento
  • Revogação do consentimento

Como o cadastro inicial é feito pela transportadora empregadora, o primeiro canal para exercer esses direitos é o setor administrativo dela. Se preferir, você pode falar diretamente com a Forbos pelos contatos abaixo.

Todas as permissões são solicitadas no momento do uso (runtime permissions) e podem ser revogadas a qualquer momento nas configurações do sistema operacional. Revogar uma permissão desabilita apenas a funcionalidade que depende dela — o restante do App continua funcionando.

9. Menores de idade

O App é destinado exclusivamente a motoristas profissionais maiores de 18 anos. Não coletamos intencionalmente dados de menores.

10. Alterações nesta política

Podemos atualizar esta política quando houver mudanças relevantes na operação do App ou na legislação. A versão mais recente estará sempre disponível nesta mesma URL. Alterações significativas serão comunicadas via notificação dentro do App ou pelo canal da transportadora.
